磁盘操作

对磁盘的挂载分区等操作

1.先在磁盘上分区

2.创建物理卷，把新建的分区映射成物理卷

3.创建卷组

4.从卷组里面创建逻辑卷

5.安装文件系统

6.blkid 配置文件

7.mount -a

设置固定ip地址

nmcli connection modify eno16777736 ipv4.addresses 172.168.1.13/24 ipv4.gateway 192.168.1.1/24 ipv4.method manual connection.autoconnect yes

重新加载网卡

nmcli connection reload

激活网卡

nmcli connection up eno*

设置好后重启网卡服务

systemctl restart network

软件安装

rpm方式安装

​ 光盘镜像安装

​ 1.挂载光驱

​ mount /dev/sr0 /mnt

​ 2.进入光盘镜像包目录

​ cd /mnt/Packages

​ 3.相关安装命令

​ 安装

​ rpm -ivh 包名

​ 卸载

​ rpm -e 包名

​ 查看是否安装了tree

​ rpm -eq | grep tree

​ rpm -lq

[base] 源标识，可以自定义，推荐base

name=base 源名称，可以自定义，推荐base

baseurl=file:// /mnt 源地址，因为是本地源 所以是应本地文件协议file://

enable=1 启用源

gpgcheck=0 关闭校验
