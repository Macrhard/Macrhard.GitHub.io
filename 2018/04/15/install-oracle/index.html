在RedhatEnterprise6.5环境中安装Oracle 11.2g

配置本地yum源

创建本地yum库

vim /etc/yum.repos.d/yum.repo

挂载iso镜像

mount /dev/sr0

关闭SELinux

vim /etc/selinux/config

关闭防火墙

安装Vmware tools

将 压缩包解压到桌面，执行shell脚本，一路回车确定即可。
