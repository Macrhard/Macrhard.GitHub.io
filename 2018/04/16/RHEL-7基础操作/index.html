RHEL 7基础操作

基础命令

系统工作命令

  1. echo

  2. date

  3. wget

    用于在终端中下载网络文件

  4. pidof

    查询指定某个服务的PID值

  5. kill

    终止指定PID的服务进程

系统状态检测命令

  1. ifconfig

    用于获取网卡配置和网络状态信息

  2. uname

    查看系统内核与系统发行版本号

  3. uptime

    显示系统时间，已运行时间，启用终端数量，平均负载值等信息

  4. free

    显示系统中内存的使用量信息

  5. who

    查看当前登入主机的用户终端信息

  6. last

    查看所有系统的登陆记录，但该信息以日志形式保存的，已修改。

  7. history

    用于查看执行过命令记录，可以自定义/etc/profile文件中的HISTSIZE变量值，设置保存记录条数。

文本编辑命令

  1. more

    用于查看较大的文本文件

  2. head

    -n N 指定查看文档的前N行

  3. tail

    -n N指定查看文档的最后N行

  4. tr

    用于替换文本文件中的字符

  5. wc

    用于统计制定文本的行数，字符数，字节数
    -l 只显示行数
    -w 只显示字符数
    -c 只显示字节数

  6. stat

    查看文件的具体存储信息和时间等信息

  7. cut

    用于按列提取文本字符串

    -d 指定分隔符
    -f 指定截取哪一列
    —output=delimiter=mydeliniter 指定自己的分隔符

  8. diff

    用于比较多个文本文件的差异

打包压缩及搜索

  1. tar

    用打包/压缩或解压
    -v 显示压缩或解压的过程
    -f file文件 输出结果到文件或设备 必须写且写在最后
    -c 打包
    -x 解包 把文件提取出来
    -j 用gzip 压缩或解压 后缀 .gz
    -z 用bzip 压缩或解压 后缀 .bz2
    -t 查看压缩文件，不提取
    -p 保留原始的权限与属性
    -P 使用绝对路径来压缩
    -C 指定解压到到目录

    不同的系统常用的打包压缩方式是不同的
    • ｗｉｎｄｏｗｓ　常用rar
    • ｍａｃ　常用　zip
    • ｌｉｎｕｘ 常用 tar

  2. grep

    用于在文本中执行关键词搜索，并显示匹配结果。

    grep

    -b 将可在执行文件当作文本文件来搜索

    -c 仅显示找到的行数

    -i 忽略大小写

    -n 显示行号

    -v 反向选择

  3. find

    用于查找指定文件，liunx中的一切都是文件。搜索工作一般都是使用find来完成。

    find [查找的路径] 查找条件 操作

    -name 匹配名称

    -user 匹配所有者

    -group 匹配所属组

    -nouser 匹配没有所有者的文件

    -nogroup 匹配没有所有组的文件

    -exec … {}\; 后面可跟用于进一步处理搜索结果的命令

    -perm 匹配权限

    -prune 忽略某个目录

存储结构与磁盘划分

Linux系统中常见的文件系统有：Ext3 Ext4 XFS

在Linux系统中，目录、字符设备、块设备、套接字、光驱、打印机等都被抽象成了文件，“Linux系统中一些都是文件”。

常见硬件设备及其文件名称

硬件设备 文件名称
IDE设备 /dev/hd[a-d]
SCSI/SATA/U盘 /dev/sd[a-p]
打印机 /dev/lp[0-15]
光驱 /dev/cdrom[1-15]
鼠标 /dev/mouse

一块全新的硬盘需要先分区，再格式化才能挂载

挂载是使用硬件设备的最后一步。

  1. mount

    用挂载文件系统，能够把硬盘设备或分区与一个目录文件进行关联，然后就能在该目录中对设备进行存取操作了。

    mount 文件系统 挂载目录

    -a 挂载所有在/etc/fstab中定义的文件系统

    -t 指定文件系统的类型（较新的Linux系统会自动进行判断，一般不需要此参数）

    配置/etc/fstab文件，开机自动挂载文件系统设备。

    | 字段 | 意义 |
    | ———— | —————————————————————————————— |
    | 设备文件 | 一般为设备的路径+设备名称，也可以写唯一识别码UUID |
    | 挂载目录 | 指定挂载到的目录，需要挂载前创建好 |
    | 格式类型 | 制定文件系统的格式，如ext3、ext4、xfs、swap、iso9660(此为光盘设备) |
    | 权限选项 | 若设置defaults，则默认权限为：rw，suid，dev，exec，auto，nouser，async |
    | 自检 | 如为1则开机后对磁盘进行自检，为0不自检 |
    | 优先级 | 若自检字段设为1则可对多块硬盘设置优先级，不设则为0 |

如配置本地镜像yum库对/etc/fstab文件进行配置，设置自动挂载镜像到/mnt

配置自动挂载

/etc/fstab

/dev/cdrom /mnt iso9660 defaults,loop 0 0
